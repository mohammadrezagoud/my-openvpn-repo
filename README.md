# OpenVPN Auto Installer

اسکریپتی ساده برای نصب و راه‌اندازی خودکار یک **OpenVPN Server** روی سرورهای لینوکسی، که در پایان فایل اتصال کلاینت (`.ovpn`) را مستقیماً در ترمینال چاپ می‌کند.

> **نکته مهم:** برخلاف پروتکل‌هایی مثل WireGuard که یک "لینک" کوتاه برای اتصال دارند، OpenVPN از یک **فایل کانفیگ (.ovpn)** استفاده می‌کند که شامل گواهی‌ها، کلیدها و آدرس سرور است. این اسکریپت همین فایل را برایتان تولید و چاپ می‌کند.

## پشتیبانی از سیستم‌عامل‌ها

- Ubuntu 18.04+
- Debian 10+
- CentOS / Rocky Linux / AlmaLinux 8+
- Fedora

## پیش‌نیاز

- یک VPS یا سرور با دسترسی روت (Root)
- پورت باز شده در فایروال ابر (در صورت استفاده از AWS, GCP, Hetzner و ...)

## نصب سریع

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPO.git
cd YOUR_REPO
sudo bash openvpn-install.sh
```

اسکریپت چند سوال ساده از شما می‌پرسد:

1. آدرس IP عمومی سرور (به‌طور خودکار تشخیص داده می‌شود)
2. پروتکل: UDP یا TCP
3. شماره پورت (پیش‌فرض: 1194)
4. سرور DNS برای کلاینت‌ها (Cloudflare / Google / Quad9)
5. نام اولین کلاینت

در پایان نصب، محتوای فایل `.ovpn` کلاینت اول مستقیماً در ترمینال چاپ می‌شود و در مسیر زیر هم ذخیره خواهد شد:

```
~/openvpn-clients/<client-name>.ovpn
```

## مدیریت کلاینت‌ها

بعد از نصب اولیه، اگر دوباره اسکریپت را اجرا کنید یک منو نمایش داده می‌شود:

```bash
sudo bash openvpn-install.sh
```

گزینه‌ها:
1. اضافه کردن کلاینت جدید
2. حذف (لغو دسترسی) یک کلاینت
3. حذف کامل OpenVPN از سرور
4. خروج

## دانلود فایل کانفیگ از سرور به کامپیوتر خودتان

```bash
scp root@YOUR_SERVER_IP:~/openvpn-clients/<client-name>.ovpn .
```

سپس فایل را در برنامه [OpenVPN Connect](https://openvpn.net/client/) وارد کنید.

## جزئیات فنی

- رمزنگاری: `AES-256-GCM` + `SHA256`
- احراز هویت اضافه با `tls-crypt`
- تولید PKI با `Easy-RSA`
- NAT و Forwarding با `iptables` (پایدار با systemd service)
- سرویس systemd: `openvpn-server@server.service`

## هشدار امنیتی

- فایل‌های `.ovpn` شامل کلید خصوصی هستند؛ آن‌ها را در جای امن نگه دارید و به اشتراک نگذارید.
- این اسکریپت را فقط روی سروری اجرا کنید که خودتان مالک/مدیر آن هستید.
- همیشه سیستم‌عامل سرور خود را به‌روز نگه دارید.
